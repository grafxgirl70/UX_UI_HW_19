My UX Skills

User Research

I use a variety of methods to help create expereinces that consider the user first. Empathy Maps, Affinity Diagrams, Priority Matrixes, Usability Tests, and more, are all a part of my toolkit when researching features of a product.   

User Interface Design

Based on what I learn through user research and testing, I create expreinces that allow the user to easily interact with products in a more efficient and understandable way.

Front End Web Development

I use HTML, CSS, Javascript and Bootstrap to build seemless,  pixel perfect code to bring designs to life.
